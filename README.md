# FoodGuide - Приложение-ассистент по питанию

Мобильное приложение, которое помогает с рецептами, подсчётом калорий и планированием питания.

## Возможности
- Рекомендации блюд с калорийностью
- Рецепты с ингредиентами и инструкциями
- Профиль пользователя с целями
- Список избранных блюд
- Чат-интерфейс для общения

## Технологии
- React Native
- Expo
- React Native Gifted Chat

## Как запустить
1. Установите Node.js
2. Установите Expo: `npm install -g expo-cli`
3. Клонируйте репозиторий
4. Установите зависимости: `npm install`
5. Запустите: `expo start`
